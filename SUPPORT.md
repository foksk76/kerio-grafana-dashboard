# Support

## What this project aims to provide

This repository aims to provide:

- reproducible Grafana provisioning;
- practical setup guidance;
- beginner-friendly documentation;
- reference dashboard configuration for Kerio Connect mail traffic in Elasticsearch.

## How to get help

Please use GitHub Issues for:

- bug reports;
- documentation gaps;
- setup problems;
- reproducibility issues;
- dashboard improvement suggestions.

## Before asking for help

Please include:

- operating system and version;
- Docker Compose version;
- Grafana version;
- Elasticsearch version;
- exact dashboard variable values;
- exact time range;
- exact command or API request you ran;
- exact error message;
- relevant logs or screenshots if safe to share.

## Best-effort support

Support is provided on a best-effort basis.

## Important

Please do not post:

- real credentials;
- real private email data;
- non-anonymized production logs;
- proprietary vendor artifacts that cannot be legally redistributed.
